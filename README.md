# Jmeter
## Prerequisites:
* Install jdk 1.8
* Download and extract Apache Jmeter 5.4.2
* Set up JMETER_HOME
## How to run this file:
* Clone the repo
* Run CMD in the root folder
* Give following command:

```

jmeter -n -t Customers_api.jmx -l log.csv -e -o Reports 

```
